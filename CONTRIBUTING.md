Getting Started
1. Fork the Repository
Click the Fork button at the top-right corner of GitHub.

2. Clone Your Fork
git clone https://github.com/<your-username>/skills-introduction-to-github.git
3. Navigate to the Repository
cd skills-introduction-to-github
4. Create a Branch
git checkout -b my-first-branch
5. Make Changes
Edit files, add notes, or complete exercises.

6. Commit Your Changes
git add .
git commit -m "Complete GitHub learning exercise"
7. Push Your Branch
git push origin my-first-branch
8. Open a Pull Request
Submit your changes for review through GitHub.
