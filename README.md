## Overview

This repository serves as a hands-on introduction to GitHub and modern software collaboration practices.

Whether you're completely new to GitHub or looking to strengthen your understanding of open-source workflows, this project provides practical exercises and examples to help you learn by doing.

---

## Learning Objectives

By working through this repository, you will learn how to:

* Create and manage repositories
* Clone repositories locally
* Create and switch branches
* Commit changes effectively
* Push code to GitHub
* Open and review Pull Requests
* Manage Issues
* Collaborate with contributors
* Understand open-source workflows
* Follow GitHub best practices

## Getting Started

### 1. Fork the Repository

Click the **Fork** button at the top-right corner of GitHub.

### 2. Clone Your Fork

```bash
git clone https://github.com/<your-username>/skills-introduction-to-github.git
```

### 3. Navigate to the Repository

```bash
cd skills-introduction-to-github
```

### 4. Create a Branch

```bash
git checkout -b my-first-branch
```

### 5. Make Changes

Edit files, add notes, or complete exercises.

### 6. Commit Your Changes

```bash
git add .
git commit -m "Complete GitHub learning exercise"
```

### 7. Push Your Branch

```bash
git push origin my-first-branch
```

### 8. Open a Pull Request

Submit your changes for review through GitHub.
